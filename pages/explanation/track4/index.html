解説

教会旋法（モード）の変化を意識した曲。

  • Aパート
    • 251と同じ4度の進行だが、最後の2つを除く和音のモードをLydianに統一することで（FM7 → B♭M7 → E♭M7 → A♭M7 → D♭M7 → G♭M7）、進行の安定感と響きの浮遊感の両立を図る
    • 51より41の動きを意識（B♭ → F）
  • Bパート
    • Aパートとは異なり、最後以外の和音のモードをAeolian（Lydianとはほぼ真反対のモード）に統一することで（Fm7 → B♭m7 → E♭m7 → A♭m7 → D♭m7 → G♭m7）閉鎖感を演出
    • ピアノで駆け上がる部分で緊張感を出すために、augの響きを利用（Aaug(9)/D）
    • Cパート直前に登場するピアノ単独で演奏する部分は、Bill EvansのI Got It Bad And That Ain't Goodをオマージュ
  • Cパート（ルートがDの時の例）
    • Lydian：E/D
    • Ionian：A7/D
    • (Mixo-lydian：D　実際にはDM7を演奏)
    • Dorian x2：G7/D → CM7/D
    • Aeolian：B♭M7sus2/D
    • (Phrygian~Locrian：E♭/D → A♭M9/D)
    • Locrianを意識したが役割はM3下のMixo-lydianぽい（スケールは等価）：B♭7/D
    • 半音下のLydian：E♭/D♭
    • …
    • このようにモードが徐々に変化。Locrianと半音下のLydianは繋がっている。半音1-2個ずつ変化してるので、ハーモニーの雰囲気・明暗が滑らかに移り変わる
    • モードを分数コードとして解釈すると、上で書いたように分子が4度ずつずれていくことが分かった

曲のタイトルのtrは、Post251のテーマに基づいて選んだ単語transcendの最初の2文字。
