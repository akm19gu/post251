解説

曲のタイトルのtrは、Post251のテーマに基づいて選んだ単語transcendence（超越・脱却）の最初の2文字を表しています。 この曲は、3つのパート（Aパート、Bパート、Cパート）から成ります。 それぞれのパートは、教会旋法とその変化を以下のように意識して作りました。

Aパート

冒頭の和音進行は、Lydianモードに統一したものになっています。 一般的にジャズの楽曲で登場する動きはⅱ-Ⅴ7-Ⅰであり、教会旋法で解釈するとⅡ Dorian → Ⅴ Mixo-lydian → Ⅰ Ionianとなります。 今回の曲では、ルートの動きは変えず、教会旋法をLydianで統一しました（Ⅱ Lydian → Ⅴ Lydian → Ⅰ Lydian）。 Track3の解説で説明した通り、和音とスケールには対応関係があります。 ここでは、LydianをM7のqualityと解釈したので、結果として上図の和音進行が出来上がりました。 これにより、ルートの動きがⅡ-Ⅴ-Ⅰであることによる進行の安定感と、Lydianによる響きの浮遊感が両立することを期待しています。

また、最後はC7 → FM7と進行するのがジャズでは王道ですが、今回はそれを避けて変終止（Bb → FM7）を採用しました。 これにより、より柔らかくスムースな曲調を演出しました。

Bパート

Aパートとは異なり、和音のモードをAeolian～Dorian（Lydianとは真反対）に統一することで、閉鎖感を演出しました。

また、ピアノで駆け上がる部分で緊張感を出すために、augの響きを利用しました。 Daug/C（＝D/C7）は個人的に好きな和音の一つです。 Ahmad JamalのI Love Musicや、Hakos BaelzというVTuberのオープニング曲（YouTube）で使われているのを聞いて、虜になりました。

Cパート直前に登場するピアノ単独で演奏する部分は、Bill EvansのI Got It Bad And That Ain't Goodをオマージュしています。

Cパート

教会旋法の五度圏上での対応関係と明暗

教会旋法の調号は連続的に変化します。 例えば、LydianからIonianにするにはファ♯を半音下げてファに変えれば良く、IonianからMixo-lydianにするにはシを半音下げてシ♭にすれば良いです。 この連続的な変化は、五度圏を用いて説明することもできます。 例えば、C lydianの構成音は五度圏上でC, G, D, A, E, B, F♯です。 C lydianからC ionianに変化させるためには、Bの左隣にあるF♯を取り除いて、Cの左隣にあるFを追加すればいいのです。 このように、だんだん五度圏上の範囲をシフトさせていくことで、教会旋法の変化を説明することができます。 LydianからIonian、Mixo-lydian、…に変化する方向のシフトは、五度圏上では左回りになります。 Track3の解説で説明した通り、五度圏で四度上行の方向は暗いと考えられるので、Lydian → Ionian → Mixo-lydian → …となるにつれてモードの雰囲気がだんだん暗くなると解釈できます。

教会旋法の連続的な変化を応用した和音進行

ここでは、教会旋法が徐々に変化し、その変化が途切れることなく連続であるような和音進行を提案します。 例えばルートをDとすると、下記のような進行になります。

  • E7/D: Lydianとして（Dをルートとする教会旋法の中では）一意に解釈可能
  • A7/D: Ionianとして一意に解釈可能
  • D7: Mixo-lydianとして一意に解釈可能
  • G7/D: Dorianとして一意に解釈可能
  • C7/D: Aeolianとして一意に解釈可能
  • F7/D: Phrygianとして一意に解釈可能
  • B♭7/D: Locrianとして一意に解釈可能
  • （E♭7/D♭: D♭ lydianとして一意に解釈可能）

B♭7/D → E♭7/D♭は、レを半音下げてレ♭にするだけで連続的に変化できます。 つまり、Locrianと半音下のLydianは連続的に繋がっているのです。

また興味深いことに、上記の分数和音の分子に注目すると、四度ずつ上行していることが分かります。 分子に注目することでも、Lydian → Ionianの方向が暗くなると解釈することが適切であることが分かります。

なお、7thを調号が等価な長三和音で置き換えると、一意な解釈性は無くなりますが和音表記はシンプルになります。 例えば、A7/DをA/Dとすると、LydianかIonianのいずれかとして解釈可能になります。 ちなみに、E7/DをE/Dとすると、LydianかSUHMM Lydian（Track3の解説参照）として解釈できます。

今回作成した曲のCパートでは、上記の和音進行を基に少し派生させた進行を用いました。

追記：A♭M9/Dと書いたのはミスで、本当はB♭7/Dです。スケール的には等価です。
