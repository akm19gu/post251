解説

超緑（Hypergreen）とは、赤色を知覚する錐体細胞があるせいで通常は見られない、鮮やかすぎる緑色のことです。 赤色を見続けてこの細胞を疲労させた後に緑色を見ることで、一時的に超緑を知覚できるらしいです。

緑といえば個人的にはFMajが一番しっくりくるのですが、鮮烈さはシャープ系のハーモニーで表現したいと考えました。 FMajとDmはrelative key（同じ調号）なので、今回はDmを軸として、アクセントでシャープ系を鳴らすことにしました。

他にも、超緑の眩しすぎるほどの鮮烈さを表現するために、主に以下の4つを意識しました。

1. 調律を高め（A=445Hz）に設定

調律を高めにすると明るく進歩的な雰囲気に、低めにすると暗く退廃的な雰囲気になる気がしています。 この曲においてはA=445Hzがいい塩梅に感じました。

2. 冒頭にシャープ系の四度堆積の和音を使用

3. BパートからAパートに戻る際の和音進行にて、♯♭の個数の変化を意識

各和音に対応する調号は以下の通りです。

  • F/G, G/A: ド、レ、ミ、ファ、ソ、ラ、シ
  • A/B: ド♯、レ♯、ミ、ファ♯、ソ♯、ラ、シ
  • B/C♯: ド♯、レ♯、(ミ♯)、ファ♯、ソ♯、ラ♯、シ
    ここでのB/C♯にミ♯を含めると、四度堆積の浮遊感が薄れて濁りが出てしまう気がする
  • C/D: ド、レ、ミ、ファ♯、ソ、ラ、シ
  • D/E: ド♯、レ、ミ、ファ♯、ソ♯、ラ、シ
  • Gm/F, B♭M7: ド、レ、ミ、ファ、ソ、ラ、シ♭

4. 最後に15th（短9度）の和音を使用

  • E♭M7: この和音の直前までE7（Aの調におけるドミナント）で引っぱり、Aに解決すると思わせて、 Aとは（五度圏上で）真反対のE♭をルートに据えた和音E♭M7を持ってくることで、がらっと色を変える狙いがあります。
  • G/D → EM7/DM7 → A: G → D → AというⅦ♭-Ⅳ-Ⅰの動き（Ⅱ-Ⅴ-Ⅰとは反対の動き）の派生です。EM7/DM7についての詳細は後述します。
  • C♯/BM7: 最後のアルペジオ部分です。最初の音はG♯ですが、実は(C#M7/BM7)/AM7という分数和音を意識しているので、C♯/BM7という表記にしています。 こちらについても詳細は後述します。

4.1 前提知識

4.1.1 五度圏と明暗

長和音は明るい、短和音は暗いというのが通説かと思います。 一方、五度圏の五度上行の方向（C → G → D → …）が明るく、反対の方向が暗いという考え方もあります（Jacob Collierの考え方）。 つまり、五度上行の方向に音を重ねた和音が明るく、逆も同様と考えることができます。 例えば、ド・ソ・レ・ラ・ミ・シ・ファ♯を構成音とする和音CM13(♯11)は、明るいと言われれば確かに明るいです。 一方、ド・ファ・シ♭・ミ♭・ラ♭を構成音とする和音（いい表記が思い浮かばず）は、暗いと言われれば確かに暗いです。 前者の方が明るく感じるのは、ドの整数次倍音のうちパワーの強いピッチにソやミ（に近いピッチ）が含まれてきれいに響くことや、 アルペジオで鳴らしたときに音程が広い方が早く高音域に達し、比較的少ない音数でより広い音域を網羅することで生まれるある種の解放感があるからだと考えています。

和音進行についても同様で、D → G → CというⅡ-Ⅴ-Ⅰの動きよりも、B♭ → F → CというⅦ♭-Ⅳ-Ⅰの動きの方が明るいと解釈することができます。

4.1.2 Super-Ultra-Hyper-Mega-Meta Lydian

Super-Ultra-Hyper-Mega-Meta Lydianスケール（ここではSUHMM Lydianと略します）は、Jacob Collierによって普及した概念です。 上図のように、Lydianスケールの最初の5音を重ね積みあげることで出来上がります。

4.1.3 和音とスケール

和音は、スケールからいくつかの音を抜き出したものと考えることができます（つまり、スケールと和音は1対多の関係にあります）。 例えば、C lydianから作れる和音として、C, C5, C6, C6(add9), CM7, CM9, CM13(♯11)などがあります。

SUHMM Lydianから和音を作るとどうなるでしょうか？ SUHMM Lydianは12平均律のピッチすべてを含むので、すべての和音を作ることができます。 そのうち、例えばC lydian（ドレミファ♯ソ）とG lydian（ソラシド♯レ）の部分の合計9音などの近傍に注目すれば、 例えばDM7/CM7という和音が作れます。 この和音の構成音はド、ミ、ソ、シ、レ、ファ♯、ラ、hiド♯で、CM13(♯11)に♯15thの音を足して自然に拡張した和音としても解釈できます。 前述の五度圏の考え方で解釈すると、もはや明るいを通り越して明る"すぎる"感じがします。 なお、DM7/CM7には、ドとhiド♯という短9度の強烈な不協和音程が登場します。

4.2 EM7/DM7について

Aで解決する直前にEM7/DM7を用いた理由は、新しい解決方法を模索していたからです。

多くの楽曲では、三全音による緊張をドミナントモーションで解決しています。 例えば、G7 → Cというドミナントモーションでは、G7の中に含まれるシとファを、最も近くにあるstable toneであるドとミに解決します。 三全音に基づくドミナントモーションの解決はとても心地よいですが、予定調和的で時には単調さを感じることがあります。

そこで、短9度による緊張をサブドミナントモーションで解決する方法を提案します。 ここでは例として、C → Gというサブドミナントモーションの派生としてDM7/CM7 → Gという進行を考えます。 SUHMM Lydianに由来する和音であるDM7/CM7は、短9度の音程を含んだ明るすぎる和音なので、これをサブドミナントの和音として採用しています。 このとき、DM7/CM7に含まれるドとhiド♯は、最も近くにあるstable toneであるシとレに解決します。 このようにして、短9度による緊張はサブドミナントモーションで解決します。 先ほどの五度圏の考え方を用いると、ドミナントモーションで解決するのに比べて、ここで提案したサブドミナントモーションによる解決はより明るいと解釈できます。 加えて、提案した解決法の原理は、前述のドミナントモーションの解決原理と対応づいているので、音楽理論的にも適切であると考えられます。

今回の曲の最後では、EM7/DM7からAに解決していました。 ここでは、EM7/DM7に含まれるレとhiレ♯の短9度の緊張を、サブドミナントのDに対するトニックであるAに含まれるド♯とミに解決することで、"より明るく"解決しているのです。

4.3 C♯/BM7について

4.1.3の「近傍」をさらに広げると、SUHMM Lydianからは(EM7/DM7)/CM7, ((F♯M7/EM7)/DM7)/CM7, …といった和音を作れます。 今回の曲の一番最後に登場するC♯/BM7のアルペジオは、この考え方を用いて、AM7の上にBM7を重ねて、さらにその上にC♯M7を重ねて…というのを意識して導入しました。 これにより、超緑の明るすぎる要素を表現しました。
