解説

超緑とは、赤色を知覚する錐体細胞があるせいで通常は見られない、鮮やかすぎる緑色。赤色を見続けてこの細胞を疲労させた後に緑色を見ることで、一時的に超緑を知覚できるらしい。

緑といえば個人的にはFMajが一番しっくりくるが、鮮烈さはシャープ系のハーモニーで表現したい。FMajとDmはrelative keyなので、今回はDmを軸として、アクセントでシャープ系を鳴らすことに。

超緑の眩しすぎるほどの鮮烈さを表現するために、主に以下の4つを意識。

  • 冒頭にシャープ系の四度堆積の和音（DM9add13）を使用。
  • BパートからAパートに戻る際の和音進行（F/G → G/A → AM7/B → BM7/C# → CM7/D → DM7/E → Gm/F → B♭M7）にて、♯♭の個数の変化を意識。
    • F/G, G/A：♮（♯♭0個）
    • AM7/B：♯4個（一気に緊張・鮮やかに）
    • BM7/C#：♯5個
    • CM/D：♯1個
    • DM7/E：♯3個
    • Gm/F, B♭M7：♭1個（ここで緩和）
  • 最後に15th（短9度）の和音を使用。
  • 調律を高め（A=445Hz）に設定。
